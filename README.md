# 🏨 Hospitality Analytics Dashboard

## 📊 Overview
This project provides a comprehensive analytical view of hospitality performance across multiple properties and time periods.  
The dataset includes booking, revenue, occupancy, and operational metrics, enabling insights into **trends, cancellations, and property-level performance**.

---

## 📂 Dataset Contents
The Excel file contains several sheets with structured data:

### 1. Executive Overview
- Occupancy %
- Realisation %
- Total Bookings
- Revenue per Available Room (RevPAR)
- Average Daily Rate (ADR)

### 2. Revenue & Booking Analysis
- Total Revenue
- Successful Bookings
- Cancelled Bookings
- No Show Bookings
- No Show %
- Successful Booking %
- Cancellation %

### 3. Operational Performance
- Property-level revenue (May, June, July)
- Occupancy %
- Total Bookings
- Total Capacity
- Realisation %
- Avg Daily Booked Room vs. Sellable Room vs. Utilized Room

### 4. Trend & Time Intelligence
- Weekly revenue comparisons
- Current vs. Previous week differences
- Occupancy % trends
- ADR and RevPAR changes
- Seasonal variations

### 5. Overall/Detailed Data
- Property-wise breakdown (Shodwe Bay, Blu, City, Exotica, Grands, Palace, Seasons)
- Monthly revenue totals
- Cancellation %
- Average Rating %

---

## 🏨 Key Metrics
- **Total Revenue (Grand Total):** `1,708,771,229`
- **Total Bookings:** `134,590`
- **Cancelled Bookings:** `33,420`
- **No Show Bookings:** `10,965`
- **Occupancy % (Overall):** `57.9%`
- **ADR (Average Daily Rate):** `12,696`
- **RevPAR (Revenue per Available Room):** `7,347`

---

## 📈 Insights
- **Occupancy Trends:** Ranged between ~44% (Shodwe Seasons) and ~62% (Shodwe Blu).
- **Revenue Leaders:** Shodwe Exotica and Shodwe Palace consistently generated the highest revenues.
- **Cancellation Rate:** ~24.8% of bookings were cancelled.
- **No Show Rate:** ~5% of bookings resulted in no-shows.
- **Seasonality:** Revenue dipped in June (`553M`) compared to May (`581M`) and July (`572M`).
